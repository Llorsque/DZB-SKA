# De Zilveren Bal Kids 2026 — Startlijst

Interactieve startlijst tool voor De Zilveren Bal SKA Kids 2026.

## Gebruik

Open `index.html` in een webbrowser — geen installatie nodig.

### Navigeren
- **Pijltje rechts (→)** of knop in de header: volgende rit
- **Pijltje links (←)** of knop in de header: vorige rit
- **Klik op een rit** in de linker kolom: spring direct naar die rit

## Opzet

- **Links**: volledige startlijst met alle 24 ritten, verdeeld over Omloop 1 en Omloop 2
- **Rechts**: huidige rit + volgende rit, elk met 3 rijders (Wit = Binnen, Rood = Midden, Geel = Buiten) en hun persoonlijke info

## Banen
| Kleur | Baan |
|-------|------|
| ⚪ Wit | Binnen |
| 🔴 Rood | Midden |
| 🟡 Geel | Buiten |

## GitHub Pages

Zet de repository op **public** en activeer GitHub Pages (Settings → Pages → Branch: main) om de tool online te hosten via `https://<gebruikersnaam>.github.io/<reponaam>/`.
